# Instruções do modulo 4 de nodejs

Essas são instruções de importações para o modulo 4 de node

### Instalando adonis

```
npm install -g @adonisjs/cli
```

### Criar a api

```
adonis new -h
adonis new (nome da api) --api-only
```

### Iniciar o servidor

```
adonis serve --dev
```

### Instalando o eslint

```
npm install -D eslint
npx eslint --init
```
